Paint

Car was painted with Ford America Shelby Acapulco Blue Metallic. This paint has been used from 1967 until 1970, mainly for Shelby variants. In 1970 it has been used by Shelby only. Ford paint code is M3077. This is not the factory color of the car.

Livery

On top of the paint there are Shelby Le Mans Dual Stripes White.

Cylinder Heads

Cylinder heads are cast iron C1AE-6090-A heads. These heads have been produced between 1961 and 1963. They have the standard 8-bolt layout for header attachment, which is compatible with the CJ 16-bolt layout. Valves are 2.02” and 1.55”. Intake port size is 2.34” x 1.34”. Exhaust port size is 1.84” x 1.28”. Chamber size is 71cc.

Carburetor

Carburetor is a Holley Street Avenger 670 in Revision 6. It injects 670cfm via 4 barrels, two primaries, two secondaries. Primaries are mechanical with a 10.5 power valve and 68 main jets. Secondaries are vacuum-initiated using a purple Moroso spring and 72 jets. Carburetor is mounted on a 2” open spacer. Choke is electrical.

Rebuilt 2020-12

Carburetor was rebuilt in December 2020. We swapped in new float needles, because the rubber seals of old ones where brittle and their debris was causing a rough idle. We swapped in new gaskets for both metering blocks and fuel bowls, because the old ones were torn. We swapped in a new primary pump, because diaphragm of the old one was already brittle. We swapped in a new vacuum diaphgram, because the old one was worn out. We swapped in a purple vacuum spring from Moroso, because the old spring was “custom cut” and we experienced a stumble when the secondaries where opening. We increased the primary power valve from 6.5 to 10.5, because we experienced a stumble off the line. We realigned and tested the electric choke. We added an air cleaner gasket, since it was missing.
